class pyforest.binary_trees.binary_tree.BinaryTree¶

Bases: abc.ABC, Generic[pyforest.binary_trees.binary_tree.NodeType]

An abstract base class for any types of binary trees.

This base class defines the basic properties and methods that all types of binary tress should provide.

root¶

The root of the tree. The default is None.

Type

Optional[NodeType]

Notes

One reason to use abstract base class for all types of binary trees is to make sure the type of binary trees is compatable. Therefore, binary tree traversal can be performed on any type of binary trees.
