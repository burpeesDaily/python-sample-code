class pyforest.bin.forest_cli.cli¶

Bases: cmd.Cmd

A CLI for operating the Forest.

do_build(line)¶

Build a binary tree.

Options: avl-tree, bst, rb-tree, threaded-bst Example: forest> build avl-tree

do_delete(line)¶

Delete an item by the given key.

Example: forest> delete 5

do_destroy(line)¶

Destroy the existing tree.

do_detail(line)¶

Show the detail of the tree.

do_exit(line)¶

Exit the application.

do_insert(line)¶

Insert a (key, data) pair. The key must be an integer.

Example: forest> insert 7 data

do_search(line)¶

Search data by a given key.

Example: forest> search 3

do_traverse(line)¶

Traverse the binary search tree.

Options: pre, in, post, rev-in Example: forest> traverse pre
